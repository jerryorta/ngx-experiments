Lollipop Chart - Interactive Controls

Use the Storybook controls panel to reshape the lollipop in real time. Toggle showStem for a dot plot, connect to join the series into a trend line, and switch the shape / orientation. Press Randomize to re-roll the scores.

Available Controls

  • Margins: top / right / bottom / left chart margins
  • Layout: orientation, marker shape, marker size, and stem baseline
  • Visibility: stem, connect (slope) line, and value labels
  • Tooltip: enable / disable hover tooltips
  • Axis Labels: X and Y axis titles
  • Marker / Stem / Label: colors and weights from the lollipop theme slice
