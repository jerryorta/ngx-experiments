Lollipop Chart - Usage Examples

Practical patterns for createLollipopChartConfig(). One preset spans the whole family: the default is a lollipop (stem + marker), showStem: false gives a dot plot, per-point valueEnd gives a dumbbell, and connect: true joins same-seriesId markers across categories into a slope chart.

1. Basic Lollipop

A stem from the baseline to an end marker per category — the defaults. It reads like a bar chart with far less ink, so a long list of categories stays legible. 

data = [
  {{ "{" }} category: 'Austin',  value: 78 {{ "}" }},
  {{ "{" }} category: 'Denver',  value: 64 {{ "}" }},
  {{ "{" }} category: 'Seattle', value: 55 {{ "}" }},
  // ...
];

config = createLollipopChartConfig({{ "{" }} data {{ "}" }});

<nge-chart [config]="config" />

2. Dot Plot (showStem: false)

Drop the stems for a bare dot plot — pure marker positions, the minimal-ink reading of the same comparison. 

config = createLollipopChartConfig({{ "{" }} data, showStem: false, showLabels: true {{
            "}"
          }});

3. Dumbbell / Span (valueEnd)

Give each row a valueEnd and the category renders two markers joined by a span segment (the baseline stem is dropped) — the classic before → after comparison. 

data = [
  {{ "{" }} category: 'Austin',  value: 1450, valueEnd: 1980 {{ "}" }},
  {{ "{" }} category: 'Seattle', value: 1900, valueEnd: 2350 {{ "}" }},
  // ...
];

config = createLollipopChartConfig({{ "{" }} data, showTooltip: true {{ "}" }});

4. Slope Chart (connect: true)

Share a seriesId across categories and set connect: true — each series' markers join into a slope line, so its rise / fall between the two years reads at a glance. 

data = [
  {{ "{" }} category: '2019', seriesId: 'Brand A', value: 32 {{ "}" }},
  {{ "{" }} category: '2024', seriesId: 'Brand A', value: 48 {{ "}" }},
  // ...
];

config = createLollipopChartConfig({{ "{" }} data, connect: true, seriesColors {{ "}" }});

5. Click to Inspect a Marker

Provide an onClick callback — each marker hands back its source datum (category, value, and any valueEnd / seriesId). 

config = createLollipopChartConfig({{ "{" }}
  data,
  onClick: (event) => inspect(event.data.category, event.data.value),
{{ "}" }});
Clicked: {{ lastClicked() }}

6. Dynamic Data With Signals

Recommended pattern: hold the data in a signal() and rebuild the config inside a computed(). Re-rolling the scores recomputes the config and the markers animate between states. 

liveData = signal<NgeLollipopDataPoint[]>(buildData());

config = computed(() => createLollipopChartConfig({{ "{" }} data: liveData() {{ "}" }}));

<nge-chart [config]="config()" />

7. Shapes & Orientation

orientation: 'horizontal' swaps categories onto the y axis, shape changes the marker glyph ('circle' / 'diamond' / 'square'), and showLabels prints each value. 

config = createLollipopChartConfig({{
            "{"
          }} data, orientation: 'horizontal', shape: 'diamond', showLabels: true {{ "}" }});
