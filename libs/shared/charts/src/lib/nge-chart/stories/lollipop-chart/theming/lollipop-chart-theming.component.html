Lollipop Chart - Theming Examples

Every color comes from the theme.lollipop slice — marker, stem, and label. Override each with a literal color or a var(--chart-*) token; the token path tracks the surrounding theme and adapts to light / dark automatically.

Default Theme

No overrides — the stem is a muted var(--chart-outline-variant) and the marker a var(--chart-primary), both from the lollipop slice.

Marker Color (theme.lollipop.marker)

Recolor the single-series glyph and give it a contrasting outline via marker.color + marker.strokeColor.

Stem Styling (theme.lollipop.stem)

Thicken and recolor the stem with stem.color + stem.width — handy for making the stem a quiet backdrop to a bold marker.

Value Labels (theme.lollipop.label)

With showLabels on, style the printed values through label.color / label.fontSize / label.fontWeight.

Semantic Tokens (var(--chart-*))

Point marker.color / stem.color at var(--chart-tertiary) / var(--chart-secondary) so the chart tracks the active theme — toggle Storybook's light / dark switch to watch it adapt.

Multi-Series Palette (theme.lollipop.marker.colors)

For a slope chart, each seriesId maps to marker.colors[index % length] — supply a palette to color the connected series.

Full Control via theme.lollipop

A dumbbell styled through the whole slice at once — marker, the span stem, and the value label.
