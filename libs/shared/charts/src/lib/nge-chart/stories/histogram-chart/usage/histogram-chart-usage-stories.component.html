Histogram Chart - Usage Examples

Practical patterns for createHistogramChartConfig(). Feed RAW observations ({{ "{" }} value {{ "}" }}[]) — the layer bins them itself via d3-array's bin() and draws one frequency bar per bin on a continuous value axis. Set mode: 'rootogram' for a Tukey hanging rootogram with a fitted normal curve.

1. Basic Histogram

Simplest usage: pass raw values and let the layer bin them into a frequency distribution (10 uniform bins by default). 

// raw observations — NOT pre-aggregated counts
data = [{{ "{" }} value: 42 {{ "}" }}, {{ "{" }} value: 55 {{ "}" }}, {{ "{" }} value: 51 {{
            "}"
          }}, /* ... */];

config = createHistogramChartConfig({{ "{" }} data {{ "}" }});

<nge-chart [config]="config" />

2. Rootogram Mode

mode: 'rootogram' fits a normal curve from the sample mean/σ and hangs each bar from that expected frequency — a bar's gap to the axis is the fit residual. The fitted expected-frequency curve is drawn behind the bars. 

config = createHistogramChartConfig({{ "{" }}
  data,
  mode: 'rootogram', // hang bars from the fitted normal curve
{{ "}" }});

3. Bin Count (granularity)

binCount controls how many uniform bins span the data extent — more bins give a finer-grained distribution. 

config = createHistogramChartConfig({{ "{" }} data, binCount: 24 {{ "}" }});

4. Explicit Domain

Pin the binning range with domain: [min, max] so several histograms share identical bin edges regardless of their own data extent. 

config = createHistogramChartConfig({{ "{" }}
  data,
  binCount: 10,
  domain: [0, 100], // fixed edges, independent of the data
{{ "}" }});

5. Per-Bin Count Labels

showLabels: true prints each bin's observation count centered above its bar. 

config = createHistogramChartConfig({{ "{" }} data, binCount: 8, showLabels: true {{ "}" }});

6. Click to Inspect a Bin

The onClick payload carries the bin ({{ "{" }} x0, x1, count {{ "}" }}), not a single datum — the interaction unit for a histogram is a bin. 

config = createHistogramChartConfig({{ "{" }}
  data,
  onClick: (event) => inspect(event.data.x0, event.data.x1, event.data.count),
{{ "}" }});

7. Dynamic Data With Signals

Recommended pattern: hold the raw observations in a signal() and rebuild the config inside a computed(). Re-rolling the sample re-bins and re-fits the rootogram curve. 

liveData = signal<NgeHistogramDataPoint[]>(makeSamples());

config = computed(() => createHistogramChartConfig({{ "{" }} data: liveData(), mode: 'rootogram' {{
            "}"
          }}));

<nge-chart [config]="config()" />
