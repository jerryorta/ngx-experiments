Histogram Chart - Theming Examples

The histogram reads four --chart-* token slots through its theme.histogram slice: bar fill (--chart-primary), the rootogram curve stroke and its per-bin nodes (both --chart-secondary), and count-label color (--chart-on-surface). Override any of them with an explicit color.

Default Theme

No overrides — bars fill --chart-primary, the fitted rootogram curve reads --chart-secondary, and count labels read --chart-on-surface.

Bar Fill (theme.histogram.bar.color)

Recolor the bins by overriding the bar fill — the --chart-primary slot — with an explicit color.

Bar Stroke & Radius

A surface-toned stroke cleanly separates adjacent bins and a corner radius rounds each bar (radius comes from the theme slice, not a preset option).

Rootogram Curve & Nodes (theme.histogram.curve / node)

In mode: 'rootogram' the fitted normal curve takes its stroke from the curve slot and the dots threading each bar top from the node slot (both the --chart-secondary slot) — here a bold red dash with matching nodes.

Label Color (theme.histogram.label.color)

With showLabels: true, the per-bin count labels read the label slot — the --chart-on-surface slot — overridable alongside font size and weight.

Full Control via theme.histogram

Compose the whole slice — bar fill / opacity / stroke / radius, the rootogram curve and its node dots, the zeroLine baseline, and the count label — in one theme object.

Multiple Charts with Different Palettes

The same distribution rendered with three bar-fill colors. Each is the same <nge-chart> component with a different theme.histogram.bar.color.
