Histogram Chart - Interactive Controls

Use the Storybook controls panel to adjust the histogram in real-time. Switch mode between histogram and rootogram, tune binCount for granularity, toggle showLabels and showTooltip, and restyle the bar / curve / node / label theme slots. Press Randomize to re-roll the sample.

Available Controls

  • Margins: chart margins
  • Layout: bin count and inter-bar gap
  • Mode: histogram (bars from the axis) vs rootogram (bars hung from the fitted normal curve)
  • Visibility: per-bin count labels, and the y = 0 residual baseline (rootogram mode)
  • Tooltip: enable / disable hover tooltips
  • Axis Labels: X and Y axis titles
  • Bar Styling: fill color, opacity, corner radius, separator stroke
  • Curve Styling: rootogram curve color and width (rootogram mode)
  • Node Styling: rootogram curve-node color and radius — 0 hides them (rootogram mode)
  • Zero Line Styling: color of the y = 0 residual baseline (when shown)
  • Label Styling: count-label color and font size (when labels are shown)
