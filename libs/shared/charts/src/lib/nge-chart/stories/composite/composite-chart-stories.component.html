Composite Charts - Layered Visualizations

Composite charts combine multiple layer types (bar, line, etc.) in a single visualization. Layers are rendered in order, with later layers appearing on top of earlier ones.

Bar Chart with Trend Lines

Combining a bar chart with multiple trend lines. The bar layer is rendered first (in back), and the line layer is rendered on top. Each bar has a unique color, and each line series has a distinct color. A custom scaleFactory ensures both layers share the same scales. 

// Bar data - each bar with unique color
barData = [
  {{ '{' }} label: 'Jan', value: 120, color: '#42A5F5' {{ '}' }},
  {{ '{' }} label: 'Feb', value: 150, color: '#66BB6A' {{ '}' }},
  // ...
];

// Line data - multiple series
lineData = [
  {{ '{' }} x: 'Jan', y: 160, seriesId: 'Target' {{ '}' }},
  {{ '{' }} x: 'Jan', y: 120, seriesId: 'Avg' {{ '}' }},
  {{ '{' }} x: 'Jan', y: 115, seriesId: 'Forecast' {{ '}' }},
  // ...
];

config: NgeChartConfig = {{ '{' }}
  base: {{ '{' }}
    margin: {{ '{' }} bottom: 50, left: 60, right: 20, top: 20 {{ '}' }},
    showXAxis: true,
    showYAxis: true,
    xAxisLabel: 'Month',
    yAxisLabel: 'Revenue ($K)',
  {{ '}' }},
  layers: [
    // Bar layer (rendered first, in back)
    {{ '{' }}
      type: 'bar',
      data: this.barData,
      barPadding: 0.3,
      barRadius: 4,
    {{ '}' }},
    // Line layer (rendered second, on top)
    {{ '{' }}
      type: 'line',
      data: this.lineData,
      curveType: 'monotone',
      showPoints: true,
      seriesColors: ['#D32F2F', '#1976D2', '#7B1FA2'],
    {{ '}' }},
  ],
  scaleFactory: (config, dimensions) => {{ '{' }}
    // Custom scale factory that works for both layers
    const labels = barData.map(d => d.label);
    const allValues = [...barData.map(d => d.value), ...lineData.map(d => d.y)];
    const maxValue = Math.max(...allValues) * 1.1;

    const xScale = scaleBand<string>()
      .domain(labels)
      .range([0, dimensions.boundedWidth])
      .padding(0.2);

    const yScale = scaleLinear()
      .domain([0, maxValue])
      .range([dimensions.boundedHeight, 0]);

    return {{ '{' }} x: xScale, y: yScale {{ '}' }};
  {{ '}' }},
{{ '}' }};

How Composite Charts Work

Layer Ordering

Layers are rendered in array order. First layer = bottom, last layer = top.

Shared Scales

All layers share the same x and y scales from scaleFactory.

Custom Scale Factory

Combine data from all layers when calculating scale domains.

Layer Independence

Each layer is configured independently with its own styles and behaviors.
