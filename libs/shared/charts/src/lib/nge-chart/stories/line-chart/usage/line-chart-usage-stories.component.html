Line Chart - Usage Examples

Practical examples showing common usage patterns for the unified chart component with line charts. Line charts support single and multi-series via the seriesId property.

1. Basic Single Line

Simplest way to create a line chart using createLineChartConfig(). Each data point has x (label) and y (value) properties. 

data = [
  {{ '{' }} x: 'Jan', y: 30 {{ '}' }},
  {{ '{' }} x: 'Feb', y: 45 {{ '}' }},
  // ...
];

config = createLineChartConfig({{ '{' }}
  data: this.data,
  showPoints: true,
  showXAxis: true,
  showYAxis: true,
  xAxisLabel: 'Month',
  yAxisLabel: 'Value',
{{ '}' }});

<nge-chart [config]="config" />

2. Multi-Series Line Chart

Hybrid series detection: Add a seriesId property to data points to automatically group them into separate lines. Use seriesColors to customize the color of each series. Series are assigned colors in order of first appearance. 

data = [
  {{ '{' }} x: 'Jan', y: 30, seriesId: 'Sales' {{ '}' }},
  {{ '{' }} x: 'Feb', y: 45, seriesId: 'Sales' {{ '}' }},
  {{ '{' }} x: 'Jan', y: 5, seriesId: 'Returns' {{ '}' }},
  {{ '{' }} x: 'Feb', y: 8, seriesId: 'Returns' {{ '}' }},
  // ...
];

config = createLineChartConfig({{ '{' }}
  data: this.data,
  seriesColors: ['#2196F3', '#4CAF50', '#FF9800'],
  showPoints: true,
  showXAxis: true,
  showYAxis: true,
{{ '}' }});

3. Area Chart

Enable showArea: true to fill the area under the line. Works great with curveType: 'monotone' for smooth curves. 

config = createLineChartConfig({{ '{' }}
  data: this.data,
  showArea: true,
  curveType: 'monotone',
  showPoints: true,
{{ '}' }});

4. Curve Types

Choose from three curve interpolation types: 'linear' (default), 'monotone' (smooth), or 'step'.

5. Click Handling

Handle point clicks by providing an onClick callback. Requires showPoints: true to enable clickable points. 

config = createLineChartConfig({{ '{' }}
  data: this.data,
  showPoints: true,
  onClick: (event) => {{ '{' }}
    console.log('Clicked:', event.data.x, event.data.y);
  {{ '}' }},
{{ '}' }});

6. Dynamic Data with Signals

Recommended pattern: Use a computed signal to rebuild the config when data changes. The chart will animate smoothly between data states. 

data = signal<NgeLineDataPoint[]>([...]);

config = computed(() =>
  createLineChartConfig({{ '{' }}
    data: this.data(),
    showPoints: true,
  {{ '}' }})
);

<nge-chart [config]="config()" />

7. Tooltip on Hover

Enable tooltips by adding tooltip: {{ '{' }} enabled: true {{ '}' }}. Tooltips appear when hovering over data points. 

config = createLineChartConfig({{ '{' }}
  data: this.data,
  showPoints: true,
  tooltip: {{ '{' }}
    enabled: true,
    position: 'follow-mouse',
  {{ '}' }},
{{ '}' }});
