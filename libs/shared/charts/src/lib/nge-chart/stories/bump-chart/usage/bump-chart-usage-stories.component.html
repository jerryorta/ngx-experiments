Bump Chart - Usage Examples

Practical examples for the bump (rank-over-time) chart. A bump chart plots how several named series rank against each other across an ordered x axis — each datum carries a metric value and the layer derives a 1..N rank per x-tick (highest value = rank 1 by default). Every point needs a seriesId.

1. Basic Usage

Simplest way to create a bump chart using createBumpChartConfig(). Each datum has x (ordered position), seriesId (the rank line it belongs to), and value (the metric driving the rank). 

data = [
  {{ "{" }} x: '2019', seriesId: 'Nova', value: 45 {{ "}" }},
  {{ "{" }} x: '2019', seriesId: 'Pulse', value: 60 {{ "}" }},
  {{ "{" }} x: '2020', seriesId: 'Nova', value: 52 {{ "}" }},
  // ...one row per (series, year)
];

config = createBumpChartConfig({{ "{" }}
  data: this.data,
  xAxisLabel: 'Year',
{{ "}" }});

<nge-chart [config]="config" />

2. Rank Order (desc vs asc)

rankOrder controls which direction the per-x-tick ranking runs. 'desc' (default) ranks the highest value as rank 1; 'asc' ranks the lowest as rank 1. Rank 1 always sits at the top.

desc — highest value = rank 1
asc — lowest value = rank 1

3. Curve Types

Choose the rank-line interpolation via curveType: 'bump' (default, symmetric S-bend), 'linear', or 'monotone'.

bump
linear
monotone

4. Points & Labels

Per-rank circles (showPoints) and end-of-line series labels (showLabels) both default on and aid reading. Turn both off for a clean lines-only rank plot. 

config = createBumpChartConfig({{ "{" }}
  data: this.data,
  showPoints: false,
  showLabels: false,
{{ "}" }});

5. Click Handling

Handle point clicks by providing an onClick callback. Requires showPoints: true (the default) to enable clickable circles. 

config = createBumpChartConfig({{ "{" }}
  data: this.data,
  onClick: (event) => {{ "{" }}
    console.log(event.data.seriesId, event.data.x, event.data.value);
  {{ "}" }},
{{ "}" }});

6. Dynamic Data with Signals

Recommended pattern: use a computed signal to rebuild the config when data changes. Ranks re-derive and the lines animate smoothly between states. 

data = signal<NgeBumpDataPoint[]>([...]);

config = computed(() =>
  createBumpChartConfig({{ "{" }}
    data: this.data(),
    xAxisLabel: 'Year',
  {{ "}" }})
);

<nge-chart [config]="config()" />

7. Tooltip on Hover

Enable tooltips by adding tooltip: {{ "{" }} enabled: true {{ "}" }}. Hovering a point shows its series, x, and derived rank. 

config = createBumpChartConfig({{ "{" }}
  data: this.data,
  tooltip: {{ "{" }}
    enabled: true,
    position: 'follow-mouse',
  {{ "}" }},
{{ "}" }});

8. Legend

Set legend: {{ "{" }} enabled: true {{ "}" }} to auto-generate a legend from the series data. The swatch color matches each series' rank line. 

config = createBumpChartConfig({{ "{" }}
  data: this.data,
  legend: {{ "{" }} enabled: true, position: 'bottom' {{ "}" }},
{{ "}" }});
