Bump Chart - Theming Examples

Default Theme

No theme provided — rank lines cycle the built-in --chart-* token palette by series index.

Custom Series Palette

Pass seriesColors to color each rank line. Series are assigned colors in order of first appearance (colors[index % length]).

Thicker, Dashed Rank Lines

Line stroke width and dash pattern via config.theme.bump.line (width, dash).

Larger Rank Points

Per-rank circle radius, hover radius, and stroke width via config.theme.bump.point.

Bold End Labels

End-of-line series label color, size, and weight via config.theme.bump.label.

Custom Axis Theme

Axis line, tick, and label colors plus typography via config.theme.axis.

Tooltip Positioning

Hover over points to see tooltips. Position is set via tooltip: { enabled: true, position: 'above' | 'below' | 'follow-mouse' }.
