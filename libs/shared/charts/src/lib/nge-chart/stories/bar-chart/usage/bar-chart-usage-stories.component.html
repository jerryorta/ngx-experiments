Bar Chart - Usage Examples

Practical examples showing common usage patterns for the unified chart component with bar charts.

1. Basic Usage

Simplest way to create a bar chart using createBarChartConfig() 

data = [
  {{ '{' }} label: 'Jan', value: 30 {{ '}' }},
  {{ '{' }} label: 'Feb', value: 45 {{ '}' }},
  // ...
];

config = createBarChartConfig({{ '{' }}
  data: this.data,
  showLabels: true,
{{ '}' }});

<nge-chart [config]="config" />

2. Click Handling

Handle bar clicks by providing an onClick callback in the config. The event contains the clicked data point. 

config = createBarChartConfig({{ '{' }}
  data: this.data,
  onClick: (event) => {{ '{' }}
    console.log('Clicked:', event.data.label, event.data.value);
  {{ '}' }},
{{ '}' }});
Last clicked: {{ lastClickedBar() }}

3. Dynamic Data with Signals

Recommended pattern: Use a computed signal to rebuild the config when data changes. This ensures the chart re-renders when data updates. 

// Signal holds the reactive data
data = signal<NgeBarDataPoint[]>([...]);

// Computed signal rebuilds config when data changes
config = computed(() =>
  createBarChartConfig({{ '{' }}
    data: this.data(),
    showLabels: true,
  {{ '}' }})
);

// Template uses the computed signal
<nge-chart [config]="config()" />

4. Horizontal Bar Chart

Set orientation: 'horizontal' for horizontal bars. Useful for category labels that are long or numerous. 

config = createBarChartConfig({{ '{' }}
  data: this.data,
  orientation: 'horizontal',
  showLabels: true,
  showYAxis: true,
{{ '}' }});

5. Statistical Lines

Enable showMeanLine and showMedianLine to display statistical reference lines across the chart. 

config = createBarChartConfig({{ '{' }}
  data: this.data,
  showLabels: true,
  showMeanLine: true,
  showMedianLine: true,
{{ '}' }});

6. Per-Bar Colors (Status Indicators)

Add a color property to individual data points to override the theme color. Useful for status indicators or categorical data. 

data = [
  {{ '{' }} label: 'Completed', value: 85, color: '#4CAF50' {{ '}' }},
  {{ '{' }} label: 'In Progress', value: 45, color: '#2196F3' {{ '}' }},
  {{ '{' }} label: 'Pending', value: 30, color: '#FF9800' {{ '}' }},
  {{ '{' }} label: 'Failed', value: 12, color: '#F44336' {{ '}' }},
];

7. Tooltip on Hover

Enable tooltips by adding tooltip: {{ '{' }} enabled: true {{ '}' }} to show data details on hover. The position option controls where the tooltip appears: 'above', 'below', or 'follow-mouse'. 

config = createBarChartConfig({{ '{' }}
  data: this.data,
  showLabels: true,
  tooltip: {{ '{' }}
    enabled: true,
    position: 'follow-mouse', // 'above' | 'below' | 'follow-mouse'
  {{ '}' }},
{{ '}' }});

8. Custom Tooltip Template

Provide a custom tooltip template using ng-template with the #ngeChartTooltip reference. The template receives the tooltip content as the implicit context variable. 

<nge-chart [config]="config">
  <ng-template #ngeChartTooltip let-content>
    <div class="my-custom-tooltip">
      <strong>{{ '{{' }} content?.label {{ '}}' }}</strong>
      <span>{{ '{{' }} content?.value | currency:'USD':'symbol':'1.0-0' {{ '}}' }}</span>
    </div>
  </ng-template>
</nge-chart>
{{ content?.label }}
{{ content?.value | currency:'USD':'symbol':'1.0-0' }}

9. Label Formatting

Use labelFormat to format value labels on bars, and yAxisTickFormat to format Y axis tick labels. Both accept a callback (value: number) => string. 

config = createBarChartConfig({{ '{' }}
  data: this.data,
  labelFormat: (v) => '$' + (v / 1000).toFixed(0) + 'K',
  showLabels: true,
  showYAxis: true,
  yAxisTickFormat: (d) => '$' + (d / 1000) + 'K',
{{ '}' }});
