Large Typography Theme

Individual colors via data[].color - overrides theme

Horizontal + Purple Theme

Horizontal orientation with custom bar radius

Custom Axis Theme

Axis colors via config.theme.axis

Statistical Lines Theme

Mean/median line colors via config.theme.bar.statistical

Hover over bars to see tooltips. Tooltip position is configured via tooltip: { enabled: true, position: 'above' | 'below' | 'follow-mouse' }
