Financial Chart - Usage Examples

Practical patterns for createFinancialChartConfig(). Feed OHLC sessions ({{ "{" }} date, open, high, low, close {{ "}" }}) and pick a variant: 'candlestick' (default) draws a wick + body per session, 'kagi' folds the close series into a reversal-driven zigzag, and 'renko' walks it into a fixed-brick staircase. Axes default ON.

1. Basic Candlestick

Simplest usage: pass the OHLC series and let the layer draw it. The body spans open→close (green when close ≥ open, red otherwise) and the thin wick spans low→high. 

data = [
  {{ "{" }} date: '2024-01-02', open: 100, high: 104, low: 99, close: 103 {{ "}" }},
  {{ "{" }} date: '2024-01-03', open: 103, high: 108, low: 102, close: 107 {{ "}" }},
  // ...
];

config = createFinancialChartConfig({{ "{" }} data {{ "}" }});

<nge-chart [config]="config" />

2. Kagi

variant: 'kagi' folds the close series into a reversal-driven zigzag; the line flips yang (thick — broke the prior shoulder) ↔ yin (thin — broke the prior waist). Time is dropped — one slot per vertex. 

config = createFinancialChartConfig({{ "{" }} data, variant: 'kagi' {{ "}" }});

3. Renko

variant: 'renko' walks the close series emitting fixed-height bricks — an ascending staircase that only reverses after a move of about 2·brickSize against the trend. 

config = createFinancialChartConfig({{ "{" }} data, variant: 'renko' {{ "}" }});

4. Candlestick With Tooltip

showTooltip: true enables the hover tooltip (candlestick only); the default formatter shows the session date and its closing price. Hover a candle to see it. 

config = createFinancialChartConfig({{ "{" }} data, showTooltip: true {{ "}" }});

5. Click to Inspect a Candle

The onClick payload carries the whole candle ({{ "{" }} date, open, high, low, close {{ "}" }}), so you can drill into the clicked session. Kagi/renko are derived transforms with no 1:1 source candle, so clicks fire for candlestick only. 

config = createFinancialChartConfig({{ "{" }}
  data,
  onClick: (event) => inspect(event.data),
{{ "}" }});
Clicked: {{ lastClicked() }}

6. Dynamic Data With Signals

Recommended pattern: hold the OHLC series in a signal() and rebuild the config inside a computed(). Randomize re-rolls a fresh, seeded random-walk series. 

liveData = signal<NgeFinancialDataPoint[]>(makeWalk());

config = computed(() => createFinancialChartConfig({{ "{" }} data: liveData() {{ "}" }}));

<nge-chart [config]="config()" />
