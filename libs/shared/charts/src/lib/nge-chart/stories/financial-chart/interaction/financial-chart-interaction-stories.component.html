Financial Chart - Interactive Controls

Use the Storybook controls panel to adjust the chart in real-time. Switch variant between candlestick, kagi and renko, tune the candleWidth / reversalThreshold / brickSize geometry, toggle the axes and the tooltip, and restyle the up / down / wick / kagi theme slots. Press Randomize to re-roll a fresh OHLC walk.

Available Controls

  • Margins: chart margins
  • Layout: variant (candlestick / kagi / renko) and candle width
  • Financial: kagi reversal threshold (absolute or % of range) and renko brick size — 0 uses the auto default (3% / 5% of the close range)
  • Visibility: X and Y axis visibility
  • Tooltip: enable / disable the hover tooltip and its position
  • Financial Styling: up / down body colors, wick color, and the kagi yang / yin line colors
