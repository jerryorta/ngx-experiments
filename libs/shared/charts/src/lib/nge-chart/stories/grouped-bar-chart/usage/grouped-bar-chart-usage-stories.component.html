Grouped Bar Chart - Usage Examples

Grouped bar charts show side-by-side bars grouped by category. Ideal for comparing metrics across groups (e.g., Active vs Closed with Avg/Min/Max).

1. Basic Grouped Bar Chart (Vertical)

CMA-style data: Active vs Closed groups, each with Avg/Min/Max bars side-by-side. Uses createGroupedBarChartConfig() with labelFormat and yAxisTickFormat. 

data: NgeGroupedBarDataPoint[] = [
  {{ '{' }} groupId: 'Active', label: 'Avg $/sqft', value: 185, color: '#4CAF50' {{ '}' }},
  {{ '{' }} groupId: 'Active', label: 'Min', value: 142, color: '#81C784' {{ '}' }},
  {{ '{' }} groupId: 'Active', label: 'Max', value: 225, color: '#2E7D32' {{ '}' }},
  {{ '{' }} groupId: 'Closed', label: 'Avg $/sqft', value: 178, color: '#2196F3' {{ '}' }},
  {{ '{' }} groupId: 'Closed', label: 'Min', value: 135, color: '#64B5F6' {{ '}' }},
  {{ '{' }} groupId: 'Closed', label: 'Max', value: 210, color: '#1565C0' {{ '}' }},
];

config = createGroupedBarChartConfig({{ '{' }}
  data: this.data,
  showLabels: true,
  labelFormat: (v) => '$' + v,
  showXAxis: true,
  showYAxis: true,
  yAxisTickFormat: (d) => '$' + d,
{{ '}' }});

2. Horizontal Grouped Bar Chart

Same data rendered horizontally with orientation: 'horizontal'. 

config = createGroupedBarChartConfig({{ '{' }}
  data: this.data,
  orientation: 'horizontal',
  showLabels: true,
  labelFormat: (v) => '$' + v,
  showXAxis: true,
  showYAxis: true,
  xAxisTickFormat: (d) => '$' + d,
{{ '}' }});

3. Grouped Bar with Tooltip

Tooltips show group + label + value on hover. Enable with tooltip: {{ '{' }} enabled: true {{ '}' }}. 

config = createGroupedBarChartConfig({{ '{' }}
  data: this.data,
  showLabels: true,
  tooltip: {{ '{' }}
    enabled: true,
    position: 'follow-mouse',
    width: 160,
  {{ '}' }},
{{ '}' }});

4. Dynamic Data with Signals

Recommended pattern: Use a computed signal to rebuild the config when data changes. Click "Randomize Data" to see animated updates. 

data = signal<NgeGroupedBarDataPoint[]>([...]);

config = computed(() =>
  createGroupedBarChartConfig({{ '{' }}
    data: this.data(),
    showLabels: true,
  {{ '}' }})
);

<nge-chart [config]="config()" />
