Area Chart - Theming Examples

Default Theme

No overrides — the three stacked series pull their colors from the theme's area.fill.colors palette (which resolves to var(--chart-*) tokens).

Top-Edge Stroke (showLine + theme.area.line.width)

Enable showLine to stroke the top edge of each band, then thicken it via theme.area.line.width for a bolder read.

Custom Axis Theme

Axis label, line, and tick colors via theme.axis.

Grid Theme

Custom gridline color, dash pattern, and width via theme.grid (with showXGrid / showYGrid on).

Fill Opacity

fillOpacity is the key legibility lever — low keeps stacked overlaps airy, high makes each band read as a denser layer.

Branded via --chart-* Token Overrides (light + dark)

The layer's default palette resolves to var(--chart-primary), var(--chart-secondary), and var(--chart-tertiary). Override those tokens on an ancestor (never --mat-sys-*) and the fills follow — no seriesColors in the config. The overrides below are defined for BOTH the light and dark Storybook themes, so switch the toolbar theme to see each set.
