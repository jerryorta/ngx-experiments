Scatter Chart - Theming Examples

Default Theme (Material Design)

No theme provided - uses Material Design CSS variables

Green Theme

Custom point colors via config.theme.scatter.point

Opacity & Stroke Theme

Control fill opacity and point outline via config.theme.scatter.point (opacity, strokeColor, strokeWidth)

Per-Point Colors

Individual colors via data[].color - overrides theme

Bubble Chart (Per-Point Sizes)

Per-point radius via data[].size encodes a third dimension

Custom Axis Theme

Axis colors via config.theme.axis with axis labels

Large Typography Theme

Custom axis font sizes via config.theme.axis

Tooltip Positioning

Hover over points to see tooltips. Tooltip position is configured via tooltip: { enabled: true, position: 'above' | 'below' | 'follow-mouse' }

Position: Above
Position: Below
Position: Follow Mouse

Custom Series Palette (seriesColors)

Theme multi-series data by passing a seriesColors array — series i uses seriesColors[i % length]. Combined with legend: { enabled: true } the chart auto-generates one legend entry per seriesId, and legend.position can move it (right, below).
