Crosshair — Theming

Same chart config, three looks. The dashed guide reads var(--chart-crosshair-guide, var(--chart-on-surface)), so a container can recolour it with one token. The shared tooltip card is an Angular template — it follows the --chart-* surface tokens, and its own crosshair-tip--* classes can override anything further. Move the pointer across each plot.

1. Default surface

No overrides — the guide inherits --chart-on-surface and the card uses the default surface tokens.

2. Midnight (token bundle)

The container overrides --chart-surface / --chart-on-surface / --chart-outline (dark surface, light text) plus a cyan --chart-crosshair-guide. The card follows the surface tokens automatically — no card-specific class needed.
