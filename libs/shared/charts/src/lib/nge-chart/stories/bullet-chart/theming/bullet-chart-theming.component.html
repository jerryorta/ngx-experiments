Bullet Chart - Theming Examples

Examples demonstrating how to customize bullet chart appearance using themes. Override colors for background bar, progress bar, progress indicator, and limit indicators.

Default Theme

The default appearance using Material Design system colors.

Color Themes

Custom color themes applied via the theme.bullet configuration.

Blue Theme
Green Theme
Purple Theme
Orange Theme 
const blueConfig: NgeChartConfig = {{ '{' }}
  ...createBulletChartConfig({{ '{' }} data: this.data {{ '}' }}),
  theme: {{ '{' }}
    bullet: {{ '{' }}
      backgroundBar: {{ '{' }} color: '#E3F2FD' {{ '}' }},
      progressBar: {{ '{' }} color: '#2196F3' {{ '}' }},
      progressIndicator: {{ '{' }} color: '#1565C0' {{ '}' }},
      limitIndicator: {{ '{' }} color: '#1976D2' {{ '}' }},
    {{ '}' }},
  {{ '}' }},
{{ '}' }};

Custom Colors via Data

Set the color property on the data point to override the default theme color. This is the recommended way to apply semantic colors (success, warning, error).

Green (Success)
Orange (Warning)
Red (Error) 
// Set color on the data point
data: {{ '{' }}
  min: 0,
  max: 100,
  progress: 85,
  color: '#00C853', // Custom green
  units: '%',
{{ '}' }}

Custom Dimensions

Customize bar height and indicator dimensions for different visual weights. 

createBulletChartConfig({{ '{' }}
  data: this.data,
  barHeight: 20,
  progressIndicatorHeight: 50,
  progressIndicatorWidth: 10,
  limitIndicatorHeight: 50,
  limitIndicatorWidth: 4,
{{ '}' }})

Transparency Effects

Use RGBA colors to create transparency effects for a softer appearance. 

theme: {{ '{' }}
  bullet: {{ '{' }}
    backgroundBar: {{ '{' }} color: 'rgba(103, 58, 183, 0.15)' {{ '}' }},
    progressBar: {{ '{' }} color: 'rgba(103, 58, 183, 0.7)' {{ '}' }},
    progressIndicator: {{ '{' }} color: '#673AB7' {{ '}' }},
  {{ '}' }},
{{ '}' }}
