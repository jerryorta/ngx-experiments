Bullet Chart - Usage Examples

Practical examples showing common usage patterns for the unified chart component with bullet charts. Bullet charts are ideal for showing progress toward a goal or comparing a value against a range.

1. Basic Usage

Simplest way to create a bullet chart using createBulletChartConfig(). Provide min, max, and progress values. 

data: NgeBulletDataPoint = {{ '{' }}
  min: 0,
  max: 100,
  progress: 75,
  units: '%',
{{ '}' }};

config = createBulletChartConfig({{ '{' }}
  data: this.data,
{{ '}' }});

<nge-chart [config]="config" />

2. With Tooltip

Enable tooltips to show the current progress value on hover. 

config = createBulletChartConfig({{ '{' }}
  data: this.data,
  tooltip: {{ '{' }}
    enabled: true,
    height: 50,
    width: 80,
  {{ '}' }},
{{ '}' }});

3. Grade Colors

Use dynamic coloring based on grade thresholds. Calculate the color based on the percentage: Failing (<70%) = Red, C Grade (70-80%) = Yellow/Orange, Good (>80%) = Green. 

getGradeColor(percentage: number): string {{ '{' }}
  if (percentage < 70) return '#F44336';      // red - failing
  else if (percentage <= 80) return '#FF9800'; // yellow - C grade
  else return '#4CAF50';                       // green - good
{{ '}' }}

data = {{ '{' }} min: 0, max: 100, progress: 75, color: this.getGradeColor(75), units: '%' {{ '}' }};
A Grade (92%)
C Grade (75%)
Failing (45%)

4. Click Handling

Handle clicks by providing an onClick callback in the config. 

config = createBulletChartConfig({{ '{' }}
  data: this.data,
  onClick: (event) => {{ '{' }}
    console.log('Clicked:', event.data.progress);
  {{ '}' }},
{{ '}' }});
Last clicked: {{ lastClickedProgress() }}

5. Dynamic Data with Signals

Recommended pattern: Use a computed signal to rebuild the config when data changes. This ensures the chart animates smoothly when progress updates. 

dynamicProgress = signal(50);

dynamicData = computed(() => ({{ '{' }}
  min: 0,
  max: 100,
  progress: this.dynamicProgress(),
  units: '%',
{{ '}' }}));

dynamicConfig = computed(() =>
  createBulletChartConfig({{ '{' }}
    data: this.dynamicData(),
    tooltip: {{ '{' }} enabled: true {{ '}' }},
  {{ '}' }})
);

<nge-chart [config]="dynamicConfig()" />
{{ dynamicProgress() }}%

6. Custom Ranges

Bullet charts support any numeric range, not just 0-100. Use the units property to display the appropriate suffix. 

data = {{ '{' }}
  min: 100,
  max: 500,
  progress: 350,
  units: ' MHz',
{{ '}' }};

config = createBulletChartConfig({{ '{' }}
  data: this.data,
  tooltip: {{ '{' }}
    enabled: true,
    formatContent: (data) => ({{ '{' }}
      label: 'CPU Speed',
      value: `${{ '{' }}data.progress{{ '}' }}${{ '{' }}data.units{{ '}' }}`,
    {{ '}' }}),
  {{ '}' }},
{{ '}' }});

7. Custom Tooltip Format

Use formatContent to customize the tooltip content. Return an object with label, value, and optional extra data. 

config = createBulletChartConfig({{ '{' }}
  data: {{ '{' }} min: 0, max: 1000, progress: 742, units: ' GB' {{ '}' }},
  tooltip: {{ '{' }}
    enabled: true,
    formatContent: (data) => ({{ '{' }}
      label: 'Storage Used',
      value: `${{ '{' }}data.progress{{ '}' }}${{ '{' }}data.units{{ '}' }}`,
      extra: {{ '{' }} remaining: `${{ '{' }}data.max - data.progress{{ '}' }} GB free` {{ '}' }},
    {{ '}' }}),
    height: 70,
    width: 140,
  {{ '}' }},
{{ '}' }});

8. Custom Dimensions

Customize the height and width of the progress bar, progress indicator, and limit indicators. 

config = createBulletChartConfig({{ '{' }}
  data: this.data,
  barHeight: 16,
  progressIndicatorHeight: 40,
  progressIndicatorWidth: 8,
  limitIndicatorHeight: 40,
  limitIndicatorWidth: 4,
{{ '}' }});
