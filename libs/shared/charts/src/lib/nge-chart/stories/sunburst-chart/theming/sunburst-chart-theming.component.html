Sunburst Chart - Theming Examples

Node colours resolve per-node color → the preset's seriesColors palette (by top-level branch index) → the theme segment.colors palette. Every descendant inherits its top-level branch colour unless it sets its own. Override theme.sunburst.segment.colors to reskin the whole cycle, or set stroke / strokeWidth / opacity for separators and transparency. Light / dark is handled by the review container.

Default Theme

No theme provided — uses the built-in --chart-* token palette, which adapts to the surrounding light / dark surface.

Palette Overrides

Recolour the branch cycle two ways: pass seriesColors to the preset, or override theme.sunburst.segment.colors. Both map the top-level branch input index to colors[index % length], so the palette cycles for any branch count.

Warm (seriesColors)
Cool (theme override) 
// Warm — via the preset's seriesColors option
const warmConfig = createSunburstChartConfig({{ "{" }}
  data,
  seriesColors: ['#E53935', '#FB8C00', '#FDD835'],
{{ "}" }});

// Cool — via a theme override
const coolConfig = {{ "{" }}
  ...createSunburstChartConfig({{ "{" }} data, innerRadius: 0.3 {{ "}" }}),
  theme: {{ "{" }}
    sunburst: {{ "{" }}
      segment: {{ "{" }}
        colors: ['#1E88E5', '#00ACC1', '#43A047'],
      {{ "}" }},
    {{ "}" }},
  {{ "}" }},
{{ "}" }};

Per-node Colors

Set color on a node to assign it a semantic colour. Descendants inherit their branch colour, and a per-node color (here on Rent) wins over the inherited branch / palette colour for that node. 

data = [
  {{ "{" }}
    label: 'Housing', color: '#5C6BC0',
    children: [
      {{ "{" }} label: 'Rent', value: 1800, color: '#26C6DA' {{ "}" }}, // overrides branch
      // ...
    ],
  {{ "}" }},
  // ...
];

Stroke Separators

A thicker segment.stroke / strokeWidth cleanly separates adjacent arcs — useful on dense sunbursts or when neighbouring branches share similar hues. 

theme: {{ "{" }}
  sunburst: {{ "{" }} segment: {{ "{" }} stroke: '#ffffff', strokeWidth: 3 {{ "}" }} {{ "}" }},
{{ "}" }}

Translucent Segments

Lower segment.opacity for a softer, layered look. 

theme: {{ "{" }}
  sunburst: {{ "{" }} segment: {{ "{" }} opacity: 0.7 {{ "}" }} {{ "}" }},
{{ "}" }}
