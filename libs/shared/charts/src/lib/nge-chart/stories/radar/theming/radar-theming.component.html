Radar - Theming Examples

Series color resolves positionally — the series index maps to the theme radar.series.colors palette (or the preset's seriesColors). Override theme.radar to reskin each part: series.colors for the polygon palette, series.fillOpacity / series.lineWidth / series.pointRadius for the fill + outline + dots, and axis / grid for the spoke + concentric-ring web. Light / dark is handled by the review container.

Default Theme

No theme provided — uses the built-in --chart-* token palette, which adapts to the surrounding light / dark surface.

Series Palette Overrides

Reskin the polygon palette with theme.radar.series.colors. The series index maps to colors[index % length], so the palette cycles for any series count.

Web Styling (spokes + grid rings)

The axis and grid slices recolor the radial spokes and concentric value rings independently of the data — tune the reference web so it recedes behind (or frames) the series.

Area Emphasis

series.fillOpacity, series.lineWidth and series.pointRadius tune the filled polygon — a denser fill, thicker outline, and larger vertex dots emphasise each series boundary.
