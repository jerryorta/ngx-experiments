Overlay Layer - Theming Examples

Overlay marks are themed through config.theme.overlay — the fitLine, band, meanLine, and limitLine slices (each backed by --chart-* tokens). Override them to recolour the annotation independently of the host line's theme.line.

Default Theme (tokens)

No overlay theme provided — the fit line uses the --chart-secondary token (a reference line distinct from the primary host series).

Recoloured Trend Line

theme.overlay.fitLine restyles the trend line — a bold dashed red reference over a muted grey host line (theme.line).

Control Chart Theming

theme.overlay.meanLine, limitLine, and band recolour the mean rule, the ±σ limit rules, and the shaded in-control zone together — teal centre with red breach limits.

Fan Chart Theming

theme.overlay.band (fillColor + fillOpacity) recolours the prediction-interval bands — the overlapping translucent fills darken toward the centre.
