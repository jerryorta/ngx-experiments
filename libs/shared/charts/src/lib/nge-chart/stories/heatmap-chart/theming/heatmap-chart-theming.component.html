Heatmap Chart - Theming Examples

The heatmap layer reads a theme.heatmap slice — cell (the sequential color ramp rampFrom → rampMid? → rampTo, plus emptyColor, radius, stroke, strokeWidth and opacity), bubble for bubble mode, and label for the per-cell value text. Every slot falls back to a --chart-* token; a named scheme overrides the ramp entirely.

Default Theme

No overrides — the cell ramp runs --chart-surface → --chart-primary, empty cells read --chart-surface-container-highest, and the separators the surface token.

Two-Stop Ramp (cell.rampFrom / cell.rampTo)

Override the ramp endpoints for a straight two-stop gradient — here a light-to-dark blue with matching value labels.

Three-Stop Ramp (cell.rampMid)

Add a rampMid to insert a midpoint color — a cream → orange → dark-red heat gradient, interpolated in HCL for a smooth transition.

Cell Styling (radius / stroke / emptyColor)

Round the cells with cell.radius, separate them with a thick white stroke, and give empty (null) cells an explicit emptyColor.

Bubble Mode Theming (theme.heatmap.bubble)

In mark: 'bubble' the circles take their fill / opacity / stroke from the bubble slot — here a purple fill with a darker outline (radius still encodes the value).
