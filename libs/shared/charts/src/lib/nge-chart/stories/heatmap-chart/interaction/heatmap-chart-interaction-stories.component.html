Heatmap Chart - Interactive Controls

Use the Storybook controls panel to adjust the heatmap in real-time. Switch mark between cell and bubble, tune the colPadding / rowPadding band gaps, pick a named scheme (or leave it blank for the token ramp), pin a domain, toggle showValues and the axes, and restyle the cell / bubble / label theme slots. Press Randomize to re-roll the grid.

Available Controls

  • Margins: chart margins
  • Layout: mark (cell / bubble), column & row band padding, and bubble max radius (bubble mode)
  • Color: named scheme (blank ⇒ token ramp) and an optional pinned color domain (min / max)
  • Visibility: per-cell values, and X / Y axis visibility
  • Tooltip: enable / disable hover tooltips and their position
  • Cell Styling: ramp endpoints (+ optional midpoint), empty-cell fill, corner radius, separator stroke width and opacity
  • Bubble Styling: bubble fill color and opacity, outline stroke and width (bubble mode)
  • Label Styling: value label color, font size and weight
