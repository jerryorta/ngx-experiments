Sparkline Chart - Usage Examples

Compact, axis-less trend marks meant to sit inline next to a number. Every variant is a thin preset over an existing layer — no new layer type: the line and multi-series sparklines reuse the line layer (with the end dot composed from the scatter layer), and the column and win-loss sparklines reuse the bar layer. Small mock datasets are held in signal()s and read through @let.

@let lineCfg = lineSparklineConfig();

1. Line Sparkline

createSparklineChartConfig() applies tight margins, a 1px stroke, points off, and a last-value dot (on by default) that marks where the series ends. Hover any point for a tooltip (tooltip: {{ "{" }} enabled: true {{ "}" }}). 

data = [
  {{ "{" }} x: 0, y: 42 {{ "}" }}, {{ "{" }} x: 1, y: 45 {{ "}" }}, /* … */ {{
            "{"
          }} x: 11, y: 68 {{ "}" }},
];

config = createSparklineChartConfig({{ "{" }} data {{ "}" }});

2. Column Sparkline

createColumnSparklineChartConfig() reuses the bar layer with tight margins, no value labels, and a small inter-column gap — a dense mini-histogram. 

data = [
  {{ "{" }} label: 'W1', value: 120 {{ "}" }}, {{ "{" }} label: 'W2', value: 165 {{ "}" }}, /* … */
];

config = createColumnSparklineChartConfig({{ "{" }} data {{ "}" }});
@let winLossCfg = winLossSparklineConfig();

3. Win-Loss Sparkline

createWinLossSparklineChartConfig() normalises each result to an equal-height mark by sign only — win, loss, or tie — coloured from the --chart-* tokens. Marks diverge from a central zero line: wins rise above it, losses drop below, ties collapse to nothing. A custom tooltip.formatContent maps each mark back to Win / Loss / Tie on hover. 

data = [
  {{ "{" }} label: 'G1', value: 1 {{ "}" }},   // win
  {{ "{" }} label: 'G3', value: -1 {{ "}" }},  // loss
  {{ "{" }} label: 'G5', value: 0 {{ "}" }},   // tie
];

config = createWinLossSparklineChartConfig({{ "{" }} data {{ "}" }});
@let multiCfg = multiSeriesSparklineConfig();

4. Multi-Series Line Sparkline

Two metrics in one cell via seriesId. Each series' end dot inherits that line's colour, so both endpoints stay distinguishable. 

data = [
  {{ "{" }} seriesId: 'Revenue', x: 0, y: 42 {{ "}" }}, /* … */
  {{ "{" }} seriesId: 'Cost', x: 0, y: 30 {{ "}" }}, /* … */
];

config = createSparklineChartConfig({{ "{" }}
  data,
  seriesColors: ['#1976D2', '#E53935'],
{{ "}" }});
@let columnZeroCfg = columnZeroLineConfig();

5. Column Sparkline with Zero Line

Columns support an opt-in zero baseline (default off — unlike win-loss). Pass showZeroLine: true to frame up vs down once values go negative — weekly net change here, with down weeks dropping below the baseline. 

data = [
  {{ "{" }} label: 'W1', value: 8 {{ "}" }}, {{ "{" }} label: 'W3', value: -6 {{ "}" }}, /* … */
];

config = createColumnSparklineChartConfig({{ "{" }} data, showZeroLine: true {{ "}" }});
