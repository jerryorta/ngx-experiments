Timeline Chart - Usage Examples

Common patterns for building timeline / Gantt charts.

1. Basic Gantt

The simplest timeline: one span per row.

data = [
  {{ "{" }} rowId: 'Design', start: '2024-01-01', end: '2024-01-22' {{ "}" }},
  {{ "{" }} rowId: 'Build', start: '2024-01-18', end: '2024-02-20' {{ "}" }},
  {{ "{" }} rowId: 'Build', start: '2024-02-22', end: '2024-03-08' {{ "}" }}, // 2nd bar, same row
];

config = createTimelineChartConfig({{ "{" }} data: this.data {{ "}" }});

<nge-chart [config]="config" />

2. On-Bar Labels

Draw the task label inside each bar.

config = createTimelineChartConfig({{ "{" }}
  data: this.data,
  showLabels: true,
{{ "}" }});

3. Milestones

Milestone items render as diamonds.

data = [
  // milestone: true renders a diamond at `start` (end is ignored)
  {{
            "{"
          }} rowId: 'Launch', label: 'Launch', start: '2024-03-18', end: '2024-03-18', milestone: true {{
            "}"
          }},
];

config = createTimelineChartConfig({{ "{" }} data: this.data, showMilestones: true {{ "}" }});

4. Tooltip on Hover

Show details on hover with a tooltip.

config = createTimelineChartConfig({{ "{" }}
  data: this.data,
  tooltip: {{ "{" }} enabled: true, position: 'follow-mouse' {{ "}" }},
{{ "}" }});

5. Click Handling

Handle span / milestone clicks via onClick.

config = createTimelineChartConfig({{ "{" }}
  data: this.data,
  onClick: (event) => {{ "{" }}
    this.lastClicked.set(event.data.label);
  {{ "}" }},
{{ "}" }});
Last clicked: {{ lastClicked() }}

6. Dynamic Data with Signals

Rebuild the config from a signal when data changes.

data = signal<NgeTimelineDataPoint[]>([...]);

config = computed(() =>
  createTimelineChartConfig({{ "{" }} data: this.data(), showLabels: true {{ "}" }})
);

<nge-chart [config]="config()" />

7. Time Axis Label + Gridlines

Label the time axis and add vertical gridlines.

config = createTimelineChartConfig({{ "{" }}
  data: this.data,
  showXGrid: true,
  xAxisLabel: 'Project Timeline',
  xAxisTickFormat: (d) => new Date(d).toLocaleDateString('en-US', {{ "{" }} month: 'short' {{
            "}"
          }}),
{{ "}" }});

8. Day-Level Timeline with Gridlines

A short-span sprint: daily ticks + vertical gridlines align each task to the day. Pass a day-granularity xAxisTickFormat — the default ~monthly tick target suits multi-month spans. 

config = createTimelineChartConfig({{ "{" }}
  data: this.sprintData, // Apr 1-8
  showLabels: true,
  showXGrid: true, // vertical gridlines
  xAxisTicks: 8, // daily granularity
  xAxisTickFormat: (d) =>
    new Date(d).toLocaleDateString('en-US', {{ "{" }} day: 'numeric', month: 'short' {{ "}" }}),
{{ "}" }});

9. Scrubbable Timeline (range-axis brush)

A year-long roadmap. The bottom axis is a full-range ruler with a draggable brush window: drag a handle to zoom the plot to that slice, or drag the window to scrub across past → future. The plot renders just the brushed window while the ruler always shows the full range. Use NgeTimelineChartTransform to own the scrub state. 

// The transform owns the scrub window + forces the range-axis brush on
transform = new NgeTimelineChartTransform({{ "{" }} data, showLabels: true {{ "}" }});

<nge-chart
  [config]="transform.config()"
  (chartGesture)="transform.onChartGesture($event)"
/>
<button (click)="transform.resetZoom()">Reset view</button>
