Diverging Bar Chart - Theming Examples

Diverging bar colors are driven by the data point's positiveColor and negativeColor. The color matching the value's sign is applied to both the bar and the value marker; when omitted, the built-in green (positive) / red (negative) defaults are used.

Default Theme

The default appearance - green for positive values, red for negative.

Color Pairs

Custom positiveColor / negativeColor pairs. Each chart below shows a positive value, so the positive color is displayed.

Blue / Orange
Teal / Rose
Purple / Amber
Indigo / Pink

    min: -100,
    max: 100,
    value: 45,

Positive vs Negative States

A single color pair renders differently depending on the value's sign - the positive color extends right, the negative color extends left.

Positive (+68)
Negative (-68)

Dark Surface

Bright, saturated colors read well on a dark background. 

data: {{ "{" }}
  min: -100,
  max: 100,
  value: 52,
  positiveColor: '#26C6DA', // cyan
  negativeColor: '#EC407A', // pink
{{ "}" }}

Transparency Effects

Use RGBA colors for a softer, translucent bar over the background track. 

data: {{ "{" }}
  min: -100,
  max: 100,
  value: 40,
  positiveColor: 'rgba(76, 175, 80, 0.6)',
  negativeColor: 'rgba(244, 67, 54, 0.55)',
{{ "}" }}

Custom Dimensions

Combine a custom color with larger bar and indicator dimensions for a heavier visual weight. 

createDivergingBarChartConfig({{ "{" }}
  data: {{ "{" }} min: -100, max: 100, value: 54, positiveColor: '#5E35B1' {{ "}" }},
  barHeight: 20,
  valueIndicatorHeight: 50,
  valueIndicatorWidth: 10,
  centerIndicatorHeight: 50,
  centerIndicatorWidth: 4,
  limitIndicatorHeight: 44,
  limitIndicatorWidth: 4,
{{ "}" }})
