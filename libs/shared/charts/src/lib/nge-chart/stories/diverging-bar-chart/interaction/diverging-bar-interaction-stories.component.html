Diverging Bar Chart - Interactive Controls

Use the Storybook controls panel to adjust the diverging bar configuration in real-time. Hover the chart (or press Randomize) to see the tooltip - toggle it with the showTooltip control.

Available Controls

  • Data: value, min, max, units, and positive / negative colors
  • Dimensions: bar height and value / center / limit indicator sizes
  • Center Label: the text in the bubble above the zero point
  • Margins: chart margins
  • Tooltip: enable / disable, position, size, and styling
