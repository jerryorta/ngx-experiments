Diverging Bar Chart - Usage Examples

Practical examples showing common usage patterns for the diverging bar chart. Diverging bars show a single value that extends from a center (zero) point toward the positive (right) or negative (left) side of a range - ideal for momentum, variance, sentiment, and "above / below target" indicators.

1. Basic Usage

Simplest way to create a diverging bar chart using createDivergingBarChartConfig(). Provide min, max, and value. A positive value extends right from the center; a negative value extends left. 

data: NgeDivergingBarDataPoint = {{ "{" }}
  min: -100,
  max: 100,
  value: 35,
  units: ' pts',
{{ "}" }};

config = createDivergingBarChartConfig({{ "{" }}
  data: this.data,
{{ "}" }});

<nge-chart [config]="config" />

2. With Tooltip

Enable tooltips to show the current value on hover. The default formatter prefixes positive values with + and appends the data point's units. 

config = createDivergingBarChartConfig({{ "{" }}
  data: this.data,
  tooltip: {{ "{" }} enabled: true {{ "}" }},
{{ "}" }});

3. Positive vs Negative

The bar direction and color follow the sign of value. Here the same range models market momentum: a Seller's Market (positive), a near-Balanced market, and a Buyer's Market (negative).

Seller's Market (+62)
Balanced (+4)
Buyer's Market (-58)

4. Custom Colors

Set positiveColor and negativeColor on the data point to override the default green / red. Whichever color matches the value's sign is used for the bar and value marker. 

data = {{ "{" }}
  min: -100,
  max: 100,
  value: 28,
  units: '%',
  positiveColor: '#1E88E5', // blue for positive
  negativeColor: '#EF6C00', // orange for negative
{{ "}" }};

5. Click Handling

Handle clicks by providing an onClick callback in the config. The event carries the full data point. 

config = createDivergingBarChartConfig({{ "{" }}
  data: this.data,
  onClick: (event) => {{ "{" }}
    console.log('Clicked:', event.data.value);
  {{ "}" }},
{{ "}" }});
Last clicked: {{ lastClicked() }}

6. Dynamic Data with Signals

Recommended pattern: use a computed signal to rebuild the config when data changes. The bar animates smoothly and flips color as the value crosses the center. 

dynamicValue = signal(30);

dynamicData = computed(() => ({{ "{" }}
  min: -100,
  max: 100,
  value: this.dynamicValue(),
  units: ' pts',
{{ "}" }}));

dynamicConfig = computed(() =>
  createDivergingBarChartConfig({{ "{" }}
    data: this.dynamicData(),
    tooltip: {{ "{" }} enabled: true {{ "}" }},
  {{ "}" }})
);

<nge-chart [config]="dynamicConfig()" />
{{ dynamicValue() }}

7. Custom Center Label

The bubble above the zero point defaults to Balanced. Override it with centerLabel to match your domain (e.g. Fair Value, Neutral, On Target). 

config = createDivergingBarChartConfig({{ "{" }}
  data: {{ "{" }} min: -50, max: 50, value: -12, units: '%' {{ "}" }},
  centerLabel: 'Fair Value',
{{ "}" }});

8. Custom Dimensions

Customize the height of the bar and the size of the value, center, and limit indicators for different visual weights. 

config = createDivergingBarChartConfig({{ "{" }}
  data: this.data,
  barHeight: 16,
  valueIndicatorHeight: 46,
  valueIndicatorWidth: 8,
  centerIndicatorHeight: 46,
  centerIndicatorWidth: 4,
  limitIndicatorHeight: 40,
  limitIndicatorWidth: 3,
{{ "}" }});
