Axis Grouping Tiers - Usage Examples

A second row of structure drawn beneath the tick labels — set base.xAxisGroups / base.yAxisGroups to an AxisTierConfig[] to tile the axis into labeled bands. Each example below proves one of the three band-resolution strategies (ranges, interval, groupBy) across the scale kinds that support it.

1. Linear Numeric Ranges

Explicit {{ "{" }} ranges: {{ "{" }} from, to, label {{ "}" }}[] {{ "}" }} bands on a linear scale. Boundaries are stated outright in domain units — here three even thirds of a 0-100 domain. 

config = {{ "{" }}
  ...createScatterChartConfig({{ "{" }} data, xDomain: [0, 100] {{ "}" }}),
  base: {{ "{" }}
    xAxisGroups: [
      {{ "{" }} ranges: [
        {{ "{" }} from: 0, to: 33, label: 'Low' {{ "}" }},
        {{ "{" }} from: 33, to: 66, label: 'Medium' {{ "}" }},
        {{ "{" }} from: 66, to: 100, label: 'High' {{ "}" }},
      ] {{ "}" }},
    ],
  {{ "}" }},
{{ "}" }};

2. Calendar Interval (Time Scale)

{{ "{" }} interval: 'day' | 'week' | 'month' | 'quarter' | 'year' {{ "}" }} tiles a time-scaled X axis into calendar bands derived from the visible domain — no manual boundaries. A short window ticks by week; a multi-year window automatically ticks coarser. 

weekly = {{ "{" }}
  ...createLineChartConfig({{ "{" }} data: sixWeeksOfData, showXAxis: true {{ "}" }}),
  base: {{ "{" }} xAxisGroups: [{{ "{" }} interval: 'week' {{ "}" }}] {{ "}" }},
{{ "}" }};

threeYears = {{ "{" }}
  ...createLineChartConfig({{ "{" }} data: threeYearsOfData, showXAxis: true {{ "}" }}),
  base: {{ "{" }} xAxisGroups: [{{ "{" }} interval: 'month' {{ "}" }}] {{ "}" }},
{{ "}" }};
~6-week domain → interval: 'week'
~3-year domain → interval: 'month'

3. Custom Ranges on a Continuous Scale

ranges bands don't need to be uniform width — group a numeric domain into arbitrary, non-uniform brackets (e.g. price bands) the same way as Example 1. 

config = {{ "{" }}
  ...createLineChartConfig({{ "{" }} data: homesByPrice, showArea: true, showXAxis: true {{ "}" }}),
  base: {{ "{" }}
    xAxisGroups: [
      {{ "{" }} ranges: [
        {{ "{" }} from: 0, to: 250000, label: '$0–250K' {{ "}" }},
        {{ "{" }} from: 250000, to: 500000, label: '$250K–500K' {{ "}" }},
        {{ "{" }} from: 500000, to: 1000000, label: '$500K–1M' {{ "}" }},
      ] {{ "}" }},
    ],
  {{ "}" }},
{{ "}" }};

4. Band-Scale Category Grouping

{{ "{" }} groupBy: (category) => string {{ "}" }} coalesces adjacent categories on a band scale that share a group value into one band — here 12 month categories fold into 4 quarters. 

config = {{ "{" }}
  ...createGroupedBarChartConfig({{ "{" }} data: monthlySales, showXAxis: true {{ "}" }}),
  base: {{ "{" }}
    xAxisGroups: [{{ "{" }} groupBy: quarterOfMonth {{ "}" }}],
  {{ "}" }},
{{ "}" }};

5. Y-Axis Range Tiers + Y-Axis Label

Tiers aren't X-only — set base.yAxisGroups to tile the Y axis into labeled bands. Because the rotated base.yAxisLabel and the tier stack share the left margin, the title seats just beyond the outermost tier so it clears the bands instead of overprinting them. 

config = {{ "{" }}
  ...createScatterChartConfig({{ "{" }} data, xDomain: [0, 100], yDomain: [0, 100] {{ "}" }}),
  base: {{ "{" }}
    yAxisLabel: 'Score',
    yAxisGroups: [
      {{ "{" }} ranges: [
        {{ "{" }} from: 0, to: 33, label: 'Low' {{ "}" }},
        {{ "{" }} from: 33, to: 66, label: 'Mid' {{ "}" }},
        {{ "{" }} from: 66, to: 100, label: 'High' {{ "}" }},
      ] {{ "}" }},
    ],
  {{ "}" }},
{{ "}" }};

6. Pill / Bracket Render Style

Set style: 'pill' on any tier to swap the default full-height tint + separators for an open-top bracket per band — a baseline with end ticks and a centered rounded-pill badge around the label. Same bands as Example 1; only the render style differs. 

config = {{ "{" }}
  ...createScatterChartConfig({{ "{" }} data, xDomain: [0, 100] {{ "}" }}),
  base: {{ "{" }}
    xAxisGroups: [
      {{ "{" }}
        style: 'pill',
        ranges: [
          {{ "{" }} from: 0, to: 33, label: 'Low' {{ "}" }},
          {{ "{" }} from: 33, to: 66, label: 'Medium' {{ "}" }},
          {{ "{" }} from: 66, to: 100, label: 'High' {{ "}" }},
        ],
      {{ "}" }},
    ],
  {{ "}" }},
{{ "}" }};
