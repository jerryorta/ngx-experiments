Gauge - Usage Examples

Practical examples for the gauge (single-value meter) layer. One primitive fans out across the meter family via two options: shape picks the form ({{ "{" }} 'arc' | 'linear' {{ "}" }}) and, for an arc, indicator picks the readout ({{ "{" }} 'fill' | 'needle' {{ "}" }}). A gauge renders ONE datum against its own [min, max] range — geometry is self-computed from the container, so the cartesian axes stay off.

1. Solid Gauge (Arc + Fill)

The default gauge — shape: 'arc' with indicator: 'fill' grows a filled value arc over the 270° speedometer sweep. Just pass the single data point. 

data = {{ "{" }} min: 0, max: 100, value: 72, units: '%' {{ "}" }};

config = createGaugeChartConfig({{ "{" }} data {{ "}" }});

<nge-chart [config]="config" />

2. Angular Gauge (Arc + Needle)

indicator: 'needle' swings a needle to the value instead of filling the arc — the classic dial gauge. The track stays as the full-sweep background. 

config = createGaugeChartConfig({{ "{" }}
  data: this.data,
  indicator: 'needle',
{{ "}" }});

3. Progress Bar (Linear)

shape: 'linear' draws a horizontal progress bar instead of an arc — the same value / range, laid out as a rail + fill. indicator is ignored for linear. 

config = createGaugeChartConfig({{ "{" }}
  data: this.data,
  shape: 'linear',
{{ "}" }});

4. Threshold Bands

thresholds paint ascending colored zones along the track — each entry's value is the band's upper bound (read low → high). The needle / fill reads over the top so you can see which zone the value lands in. 

config = createGaugeChartConfig({{ "{" }}
  data: this.data,
  indicator: 'needle',
  thresholds: [{{ "{" }} value: 50 {{ "}" }}, {{ "{" }} value: 80 {{ "}" }}, {{ "{" }} value: 100 {{
            "}"
          }}],
{{ "}" }});

5. Custom Sweep (Half Gauge)

startAngle / endAngle (radians, 0 = 12 o'clock, clockwise) reshape the arc — here a 180° half gauge. innerRadius is a ratio (0–1) of the self-computed outer radius, so it stays resize-safe. 

config = createGaugeChartConfig({{ "{" }}
  data: this.data,
  startAngle: -Math.PI / 2, // 9 o'clock
  endAngle: Math.PI / 2, //   3 o'clock
  innerRadius: 0.55,
{{ "}" }});

6. Tooltip

tooltip: {{ "{" }} enabled: true {{ "}" }} shows the label + value (with units) on hover, placed above the gauge center. Provide formatContent for a custom readout. 

config = createGaugeChartConfig({{ "{" }}
  data: this.data,
  tooltip: {{ "{" }} enabled: true {{ "}" }},
{{ "}" }});

7. Click Handling

Provide an onClick callback in the config. The event carries the full gauge data point, so you can react to a selection. 

config = createGaugeChartConfig({{ "{" }}
  data: this.data,
  onClick: (event) => {{ "{" }}
    console.log('Clicked:', event.data.value);
  {{ "}" }},
{{ "}" }});
Last clicked: {{ lastClicked() }}

8. Dynamic Data with Signals

Drive the gauge from a signal and rebuild the config with computed(). Re-rolling the value animates the needle swing / arc growth to its new position. 

dynamicValue = signal<NgeGaugeDataPoint>(this.data);

dynamicConfig = computed(() =>
  createGaugeChartConfig({{ "{" }} data: this.dynamicValue(), indicator: 'needle' {{ "}" }})
);

<nge-chart [config]="dynamicConfig()" />
