Gauge - Theming Examples

Every part of the gauge is themeable via theme.gauge: value for the filled arc / fill, track for the unfilled background, needle for the angular-gauge needle, threshold.colors for the ascending band palette, and label for the center readout. Colors default to the --chart-* tokens. Light / dark is handled by the review container.

Default Theme

No theme provided — uses the built-in --chart-* token palette, which adapts to the surrounding light / dark surface.

Value + Track Palette

Recolor the filled value arc with theme.gauge.value.color and the unfilled background with theme.gauge.track.color to rebrand the meter.

Needle Styling (Angular Gauge)

On an angular gauge (indicator: 'needle'), theme.gauge.needle.color and needle.width restyle the needle independently of the value arc color.

Threshold Palette

theme.gauge.threshold.colors recolors the ascending zone bands — the band index maps to colors[index % length], so the palette cycles for any band count.

Center Label Styling

theme.gauge.label restyles the center value readout — color, fontSize, and fontWeight (the font family is inherited by default).

One Theme, Three Shapes

The same theme.gauge slices apply across every catalog variant — the Solid Gauge, Angular Gauge, and Progress Bar all read the value + track (+ needle) colors the same way.
