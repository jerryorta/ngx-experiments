Radial Bar - Theming Examples

Fill colors resolve per-datum color → the preset's seriesColors palette → the theme bar.colors palette. Override theme['radial-bar'] to reskin each mark: bar.colors for the arc palette, cell.color / cell.minOpacity for the heatmap intensity ramp, and area.fillOpacity / area.lineWidth for the radial area. Light / dark is handled by the review container.

Default Theme

No theme provided — uses the built-in --chart-* token palette, which adapts to the surrounding light / dark surface.

Bar Palette Overrides

Reskin the arc palette with theme['radial-bar'].bar.colors. The datum index maps to colors[index % length], so the palette cycles for any category count.

Cell Intensity (Circular Heat Map)

The cell mark encodes value as fill opacity of a single base color. Set cell.color for the hue and raise cell.minOpacity to keep the faintest cells legible.

Area Styling

The area mark tunes the radial area fill + outline: a denser area.fillOpacity and a thicker area.lineWidth emphasise the series boundary. Series colors still come from the shared bar palette by series index.
