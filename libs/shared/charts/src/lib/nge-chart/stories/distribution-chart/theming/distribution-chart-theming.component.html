Distribution Chart - Theming Examples

The distribution layer reads a rich theme.distribution slice — box, median, mean, outlier and whisker for box mode, violin for KDE mode, and point (with a per-category colors palette) for point mode. Every group falls back to a --chart-* token; override any of them with an explicit color.

Default Theme

No overrides — the box fills a translucent --chart-primary, the median reads --chart-secondary, the mean glyph --chart-tertiary, outliers --chart-error, and whiskers the muted --chart-on-surface-variant.

Box, Median & Whisker (theme.distribution.box / median / whisker)

Recolor the three box-mode slots together — an explicit green box body, a dark-green median line, and a slate whisker — for a fully bespoke box-and-whisker.

Violin Color Scheme (theme.distribution.violin)

In render: 'violin' the KDE body takes its fill / opacity / stroke from the violin slot; here a purple density with a light inner-box median that stays legible against the fill.

Points Palette (theme.distribution.point.colors)

A per-category point.colors palette gives each class's beeswarm its own hue — the category index maps to colors[index % length].

Horizontal + Themed

Theming is orientation-agnostic: a warm amber box laid out horizontally (orientation: 'horizontal') with a bold median and matching mean glyph.

Same Data, Three Render Modes

The identical four distributions rendered box vs violin vs points, each with a coordinated teal theme.distribution slice — one <nge-chart> component, three render modes.
