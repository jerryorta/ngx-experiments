Distribution Chart - Interactive Controls

Use the Storybook controls panel to adjust the distribution in real-time. Switch render between box, violin and points, flip orientation, pick a whiskerStat, toggle showBox / showOutliers / showMean / showInnerBox, tune the jitter strategy, and restyle the box / violin / point theme slots. Press Randomize to re-roll every endpoint's samples.

Available Controls

  • Margins: chart margins
  • Layout: render mode (box / violin / points), orientation, and box width
  • Box: whisker statistic (IQR / min–max / ±σ / ±SE), and box / outlier / mean visibility (box mode)
  • Violin: inner box, KDE kernel and resolution (violin mode)
  • Points: jitter strategy, jitter width and marker radius (points mode)
  • Tooltip: enable / disable hover tooltips and their position
  • Axis Labels: X and Y axis titles
  • Box Styling: box fill color and opacity (box mode)
  • Median / Whisker Styling: median line and whisker colors (box mode)
  • Violin Styling: violin fill color and opacity (violin mode)
  • Point Styling: point fill color and opacity (points mode)
  • Outlier Styling: outlier point color (box mode)
