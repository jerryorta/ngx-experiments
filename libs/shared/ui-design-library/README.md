# shared-ui-design-library

Go-forward shared UI design library for the ngx-experiments workspace — reusable
`dlc-` components, directives, pipes, and utilities. This is the clean successor
to `shared-ui-design-library-deprecated`; components migrate in over time.

## Usage

Import from `@nge/ui-design-library`:

```typescript
import { /* … */ } from '@nge/ui-design-library';
```

## Contributing

Before adding a component, read the authoring guides in this directory:

- **`AGENTS.md`** — contributor notes + review checklist
- **`COMPONENT-ARCHITECTURE-BEST-PRACTICES.md`** — full component architecture guide

## Running unit tests

Run `nx test shared-ui-design-library` to execute the unit tests.
