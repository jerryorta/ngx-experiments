NgeCalendar

Reusable, themeable, theme-agnostic calendar library (@gigasoftware/calendar). This is the Storybook harness (ARCH-72) — the canvas every view + interaction story is authored on. The <nge-calendar> shell and views land in later ARCH-60 stories; the panel below is a placeholder that demonstrates the theming bridge.

Theme tokens

Each swatch reads a --nge-calendar-* token with a literal fallback, so it renders with no host theme and updates live when you switch the theme in the toolbar.
